Yarner

A project that aims to improve reading and writing as cultural habits through a platform that allows reading, writing and sharing interactive e-books.

Rafael is co-founder at Yarner, working with all the project's areas, including the research made based on it.

The project won over 20 prizes in science fairs in Brazil and USA, including prizes from companies like Intel, 3M, Oracle and GoDaddy.

Available at www.yarner.com.br

"Virada Cultural" of Things

"Virada Cultural" is an event that happens annually in many cities in Brazil. It brings arts in many forms to all the people in the city, for free. It has circus presentations, dance, musical shows and many other attractions.

This projet aims to construct design elements of Internet of Things (IoT) to this event. Those design interactions may not be available at moment, but those interactions are possible in a near future. The main idea is to use IoT to bring more social interactions through displays and other technologies.

More of the project can be seen at GitHub

PlayKids

PlayKids is a company that aims to help parents develop and educate their child while they are having fun.

It has many apps with content from videos, games and other activities to improve kids skills while being entertaining. It is one of the biggest apps for kids in the world.

Rafael worked at the development of PlayKids' content web server area, with both web front-end and back-end activities. He worked with a diverse team across the world with a global app.

Available for iOS, Android, Apple Watch and Apple TV. More details at playkids.com

Void Call

Void Call is a side scrolling shooting game.

The player is a hero from outer space which heard "The Void Call". It is described in legends as the ultimate sign to show the Universe's answers. The adventure make the hero cross many worlds while trying to survive "The Void Call" trials.

The first version was developed in Java for PC environment. It has being developed a mobile version.

Rafael designed the full project, including game storyline, characters design, art and coding.

The project was presented in 2012 to 2014 editions of "Cotuca's Open Doors", University of Campinas' Technician's School (Cotuca) event to show projects developed in the courses.

Content Management Systems

Rafael worked on the development of different kinds of Content Management Systems (CMS).

The "Arts Museum" CMS was designed to control all the art pieces from a museum. It enabled the user to store photos and information about each piece. Developed in Delphi, in 2012.

In 2012, Rafael developed a CMS for a library. Besides the default CMS behaviour for books, it had the feature to control users of the library and which book were available to be taken. Developed in Delphi.

In order to improve the control of maps for a local network MOBA game, Rafael developed a CMS. It had features to create new maps for the game using the sprites from it, create units with collisions in the map and organize and load it into the game. Developed in 2013, with Java.

Between 2015 and 2016, Rafael developed a CMS for multiple kind of content (videos, books, games, characters) to be served globally. It had many business rules and specific needs for each kind of content. It allowed metadata and information to be used in n-trees format and create localized version for each content. Developed with Ruby on Rails and JavaScript.
